# GamBallRun

Unityで制作した、ステージを傾けてボールをゴールまで運ぶ3Dアクションゲームです。

## プログラムの説明

プレイヤーはステージを傾けながらボールを操作し、頂上にあるゴールを目指します。ボールに速度を与えることが重要ですが、速度が大きすぎるとづてーじから落下してしまうため、速度の調整を行うことがゲームのポイントです。
ボールがゴール判定に入ると「Goal!!!!!!!」と表示され、ゲームが停止します。

## 使用技術

* Unity 6
* C#
* Rigidbody
* Collider / Trigger
* TextMeshPro UI

## 操作方法

- W / A / S / D または 矢印キー：ステージを傾ける

## 制作目的

Unityの物理演算、Collider、UI表示、C#スクリプトの基本を学ぶために制作しました。

## 今後の展望
Stageに適切なデザインを施す、ゴール時、落下時にサウンドを流す等を行い、ユーザーがより楽しくゲームプレイを行うことができるよう改善したいと考えています。
